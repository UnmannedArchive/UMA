# Seethrough — Clothing Brand Website

A minimal, editorial website for the Seethrough clothing brand. Dark aesthetic with premium typography and layered design.

## Run locally

Open `index.html` in your browser, or use a simple static server:

```bash
# Python 3
python -m http.server 8000

# Node.js (npx)
npx serve .
```

Then visit http://localhost:8000

## Customize

- **Images**: Replace the `.placeholder-img` divs with `<img>` tags pointing to your product/lookbook photos
- **Colors**: Edit the CSS variables in `styles.css` (`:root`)
- **Content**: Update collection names, about text, and footer links in `index.html`
